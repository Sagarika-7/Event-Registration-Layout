# Event-Registration-Layout
Event Registration Layout is a responsive web project that allows users to register for events by entering personal and event details. It features real-time cost calculation, QR-based UPI payments, multiple payment options, downloadable digital receipts, and an admin dashboard for managing registrations.
# Technologies used
HTML
-CSS
-JavaScript
## Features
- User-friendly event registration form
- Real-time cost calculation
- QR-based UPI payment integration
- Downloadable digital receipt
- Admin dashboard for tracking registrations
## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
